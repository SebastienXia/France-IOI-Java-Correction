# 1. Hello World
Un exercice très connu [Hello world](https://fr.wikipedia.org/wiki/Hello_world), qui permet d'afficher un simple message "Hello World", c'est le point d'entrée de presque tout langage informatique.
Dans cet exercice, il fallait utiliser `System.out.println()` pour pouvoir afficher du contenu à la sortie.

```Java
class Main {
   public static void main(String[] args) {
      System.out.println("Hello world!");
   }
}
```

La class `Main` est la class du fichier, le plus important c'est le `public static void main(String[] args) {}` qui est le point d'entrée c'est à dire le input du programme.
