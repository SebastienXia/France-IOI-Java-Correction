# 2. L'éclipse

Ici nous devons faire l'opération de soustraction `-` pour connaître le nombre de jours qu'il faudra patienter avant de pouvoir admirer l'éclipse :

```Java
class Main {
   public static void main(String[] args) {
      System.out.println(12581 - 11937);
   }
}
```

le nombre de jours qu'il faudra patienter avant de pouvoir admirer l'éclipse est **644 jours** !
