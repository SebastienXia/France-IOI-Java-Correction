# 4. L'algoréathlon

Afficher le nombre de kilomètres parcourus durant le premier jours, second jours et le troisième jours :

```Java
class Main {
   public static void main(String[] args) {
      int distanceKM = 2 + 34 + 6;
      System.out.print(distanceKM + " ");
      System.out.print((distanceKM*2) + " ");
      System.out.print((distanceKM*3) + " ");
   }
}
```
