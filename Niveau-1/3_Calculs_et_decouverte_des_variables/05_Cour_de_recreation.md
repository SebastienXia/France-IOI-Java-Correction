# 5. Cour de récréation

Calcule du périmètre (4c) et de la surface (c^2) de la cour de récréation : 

```Java
class Main {
   public static void main(String[] args) {
      int longeur = (17 * 5) + (7 * 2) +  5 + (2 * 2);  // longeur d'un côté 
      System.out.println(longeur * longeur);  // surface
      System.out.println(longeur * 4);  // périmètre
   }
}
```
