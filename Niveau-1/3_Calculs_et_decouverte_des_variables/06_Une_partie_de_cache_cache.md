# 6. Une partie de cache-cache

Afficher la suite de 1 jusqu'à 100 ligne par ligne et à la fin print -> J'arrive ! :

```Java
class Main {
    public static void main(String[] args) {
        for (int i = 1; i <= 100; i++) {
            System.out.println(i);
        }
    System.out.println("J'arrive !");
   }
}
```
